HomePageHeader Class

Defined in: HomePageHeader.js:91
Module: HomePageHeader

页面的导航栏部分， 在首页、直播间、录播间、直播详情页、录播详情页作为子组件插入。 包含了直播网站的logo、名字等信息，有首页、直播、录播、登录注册等选项， 点击可进入相关页面。 用户在登录后可以看到自己的个人信息并且有修改昵称、密码等功能， 若用户拥有老师权限则可以进行创建房间。
