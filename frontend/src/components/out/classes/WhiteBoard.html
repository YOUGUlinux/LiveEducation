WhiteBoard Class

Defined in: WhiteBoard.js:39
Module: WhiteBoard

实现教学区白板功能， 作为子组件插入直播间页面。 老师端可以对白板进行操作， 可以选择画笔、直线、几何图形等进行画图， 有填充、大小选择、橡皮擦、撤销等功能。 学生端不能对白板操作， 只能同步看到老师的操作。
